## Hello, and welcome to my personal site.

I created this to showcase my thoughts and personal work. Over time I have experimented with various blog platforms, and eventually decided that none of them really met my needs entirely. So I decided to create my own, based off a gatsby.js starter!

When I first purchased my domain name I was quite new to web development, but I knew I wanted a personal homepage, my own little slice of the internet. That first site was just pure HTML, which was what I knew at the time. Then as my skills progressed I moved to using a custom site build in PHP, and then I got bored of building everything, so I moved to using Blogger. Then I realized that Blogger didn't meet my needs so then I created a custom React.js site, that showed off my current interests, chatbots. That stuck around for a while, but then I wanted to have an easy way to post blog posts. So naturally I learned about Amazon Lightsail and created a Wordpress site hosted on Lightsail. I quickly realized that I wanted to have more access to customize my site, and I really didn't want to learn the Wordpress development ecosystem, and that's how I landed on using Gatsby. I started with a blog starter template, since I wanted a blog and was just getting into learning Gatsby, but quickly realized this is exactly what I was looking for.

This site is built using Gatsby, using Grommet as a component library, and hosted via AWS Amplify.

In my professional career, even though I am technically a full-stack engineer, I spend a lot of time building APIs in Node.js. So I wanted an opportunity to build something of my own and better learn React, while a pure React site would have been great, I decided to use Gatsby since it seemed like something I wanted to learn at the time. This gives me an opportunity to build in frameworks that I don't use day to day, as well as get into the details of managing hosting your own site on AWS.

That's all for now, thanks for taking the time to check this out, and hope you come back to see as I experiment more and more with this site!
